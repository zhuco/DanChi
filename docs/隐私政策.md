# DanChi 隐私政策草案

更新日期：2026-06-28

DanChi 当前版本以本地离线学习为主。词库、学习记录、收藏、笔记和设置默认保存在设备本地。

## 收集的信息

当前离线版本不要求注册账号，不主动上传学习数据。

本地保存的数据包括：

- 学习过的单词。
- FSRS 卡片状态、到期时间、稳定度、难度、今日学习清单、答题记录和复习反馈。
- 收藏词和笔记。
- 每日学习计划与发音设置。

## 权限说明

- 网络权限：为后续内容包更新、反馈、同步和服务状态预留。当前核心学习流程可离线使用。
- 备份能力：系统备份可能包含本地数据库和设置，用于设备迁移和恢复。

## 未成年人保护

DanChi 面向中考词汇学习场景，产品设计不包含强制广告、公开社交广场或排行榜 PK。商业发布前应补充监护人同意、客服反馈和注销删除流程。

## 数据删除

当前版本可通过系统“清除应用数据”删除本地学习记录。后续账号版本需要提供应用内注销与云端数据删除入口。
