# DanChi 阶段验收清单

## Android 本地闭环

- App 可以编译安装。
- 首次启动自动导入 `zhongkao_words.json`。
- 首页显示总词量、已学、收藏和今日计划。
- 首页唯一主学习入口为“开始提分”。
- 点击“开始提分”后进入 FSRS recognition 选择题队列。
- 新词第一次出现时先测试，答题后再展示单词详情页。
- 答题行为能写入 `review_logs`，信息页本身不写入复习记录。
- FSRS 能更新 `cards.state`、`dueAt`、`stability`、`difficulty`、`reps` 和 `lapses`。
- 到期复习以 `cards.dueAt` 为准，不使用固定艾宾浩斯天数表。
- 今日清单写入 `study_sessions` / `study_session_items`，再次点击“开始提分”能恢复未完成清单。
- 数据库 v12 后不再保留 `today_fsrs_session_items` 和 `word_study_records` 运行时表。
- 词典可以按英文或中文搜索。
- 收藏和笔记可以写入本地数据库。
- 设置页可调整每日新词上限、背词顺序、发音口音和语速。
- 首页不再显示自由练习、牢记模式、拼写、选义或听写入口。

## 内容

- 当前词库来源：`初中中考.txt`。
- 当前生成词条数：约 1974。
- 当前内容状态：可开发测试，未完成人工审核。
- 商业发布前必须补充内容来源、审核人、审核时间和授权说明。

## 不合格标准

- App 无法通过 `assembleDebug`。
- 首次启动后词库为空。
- “开始提分”答题后 `cards` 或 `review_logs` 无变化。
- 搜索词典出现崩溃。
- 发音失败导致页面不可继续操作。
- 设置变更重启后丢失。
