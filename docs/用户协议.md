# DanChi 用户协议草案

更新日期：2026-06-28

DanChi 提供中考英语词汇学习、FSRS 智能复习、词典查询、收藏和笔记等功能。

## 使用范围

当前版本仅用于个人学习和产品测试。词库内容来源于本地项目文件 `初中中考.txt`，商业发布前需要完成人工审核和授权核验。

当前主学习入口为“开始提分”。系统会根据 FSRS 调度结果自动合并新词、短期学习、遗忘重学和到期复习。当前版本不保留自由练习或牢记模式入口。

## 用户内容

用户在本地创建的笔记、收藏和学习记录归用户本人所有。当前离线版本不公开展示用户内容。

## 服务限制

系统 TTS 发音可能因设备语音引擎差异而表现不同。DanChi 会提供兜底播放逻辑，但不保证所有设备上的发音效果完全一致。

## 免责声明

DanChi 是学习辅助工具，不替代学校教学、考试大纲解读或教师指导。
