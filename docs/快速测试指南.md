# DanChi 快速测试指南

## 最快方式：直接安装 APK

最新 Debug APK 由 Gradle 生成：

- `E:\DanChi\app\build\outputs\apk\debug\app-debug.apk`

把 APK 发到手机，点击安装即可。若系统提示“未知来源应用”，按提示允许本次安装。

## USB 一键安装

手机准备：

1. 设置 -> 关于手机 -> 连续点击版本号，打开开发者选项。
2. 开发者选项 -> 打开 USB 调试。
3. USB 连接电脑，手机上允许 RSA 调试授权。

电脑执行：

```powershell
cd E:\DanChi
.\tools\InstallDebugApk.ps1
```

如果没有检测到设备，先运行：

```powershell
adb devices
```

看到设备状态为 `device` 后再安装。

## 重新打包

```powershell
cd E:\DanChi
.\tools\BuildDebugApk.ps1
```

脚本会执行单元测试、构建 Debug APK，并复制到：

```text
E:\DanChi\DanChi-debug.apk
```

## 5 分钟冒烟测试

1. 打开 App，首页应显示“DanChi”和“FSRS 智能提分”相关说明。
2. 首页总词量应显示约 1974。
3. 点“开始提分”，进入 recognition 选择题。
4. 题面应先展示英文单词和 4 个中文释义选项，不应先展示完整信息页。
5. 选择一个释义后，应记录答题结果，并展示单词详情页。
6. 点“下一题”，队列应继续推进。
7. 点发音按钮，系统 TTS 应能播放英文。
8. 返回词典，搜索 `water` 或中文 `安全`，应出现本地词条。
9. 收藏一个词，再去首页或统计查看收藏数量变化。
10. 在学习页写一条笔记并保存。
11. 去“我的 -> 学习计划”选择每日新词上限，退出重进后设置应保留。
12. 学习到一半返回首页，再点“开始提分”，应恢复同一份今日清单而不是重新生成。
13. 首页不应出现“拼写 / 选义 / 听写 / 牢记模式”等旧入口。

## 当前已知边界

- 当前是 Debug 测试包，应用 ID 是 `com.danchi.app.debug`。
- 当前核心功能离线可用，不需要账号。
- 主学习入口只有“开始提分”，新词和复习由 FSRS 统一调度。
- 今日清单保存在 `study_sessions` / `study_session_items`，已完成和未完成题会保留。
- 例句是开发期兜底模板，商业发布前需要替换为人工审核例句。
- 发音使用系统 TextToSpeech，不同手机发音效果会不同。
